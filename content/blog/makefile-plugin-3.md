Over the last one and a half year, the plugin got many new features:

* Shell language injection in recipes and <code>``</code> blocks
* Spellchecker support
* Variable references which allow completion, rename, and searching for usages
* Many other bug fixes

If you need to edit Makefiles, [install the plugin](https://plugins.jetbrains.com/idea/plugin/9333-makefile-support) now!
