## Proselint plugin for IntelliJ IDEA

Plugin is available for install via [official repository](https://plugins.jetbrains.com/idea/plugin/9854-proselint). Source code is available on <i class="fa fa-github" style="color: rgb(23,21,21)"></i> [GitHub](https://github.com/kropp/intellij-proselint)

Plugin integrates [Proselint](http://proselint.com/) into editor. Proselint and Python are to be installed separately. See [installation instructions](https://github.com/amperser/proselint).

{{< figure src="/blog/img/2017/proselint.png" caption="Proselint plugin in action" width="453" height="73" >}}

If there is a replacement suggested by Proselint, a quick-fix appears to apply it via `Alt+Enter`.

Feedback and pull requests are always welcome!
